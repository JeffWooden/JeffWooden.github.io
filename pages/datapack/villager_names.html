Villager Names

This datapack aims at giving villagers and golems random names. If this setting is enabled, the names given to the villagers depend on the biome they're in.\nYou can customize these names by giving your own list.

Project Page

Create your own names list

In the datapack, there is an option to toggle the use of a custom names list for each mob.\nBy default, this list contains only two names: foo and bar.\nIf you want to insert your own list of names, do it below and copy/paste the result into the wanted file.

Note that you'll need to unzip the datapack and use a file editor to modify a file content.
